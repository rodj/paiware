Library Book Checkout — Data Dictionary
========================================

Books
	Id              int               NOT NULL   PK, Identity(1,1)
	CreateDate      datetime2(0)     NOT NULL   Audit: record creation time
	Title           nvarchar(max)     NOT NULL
	Author          nvarchar(max)     NOT NULL
	Isbn            nvarchar(max)     NOT NULL

Members
	Id              int               NOT NULL   PK, Identity(1,1)
	CreateDate      datetime2(0)     NOT NULL   Audit: record creation time
	Name            nvarchar(max)     NOT NULL
	Email           nvarchar(max)     NOT NULL

Checkouts
	Id              int               NOT NULL   PK, Identity(1,1)
	CreateDate      datetime2(0)     NOT NULL   Audit: record creation time
	BookId          int               NOT NULL   FK -> Books.Id
	MemberId        int               NOT NULL   FK -> Members.Id
	CheckedOutAt    datetime2         NOT NULL
	DueDate         datetime2         NOT NULL   CheckedOutAt + 14 days
	ReturnedAt      datetime2         NULL       NULL = still checked out
